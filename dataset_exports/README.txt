Place one DHIS2 export CSV per dataset in this folder.

File name format:
<DATASET_UID>.csv

Example:
RlfDdEEZ317.csv

Supported header variants include:
- dataelement / dataElement / Data element / deUID / DE UID
- categoryoptioncombo / catoptcombo / categoryOptionCombo / Category option combo / cocUID / COC UID
- value / Value / localValue

The app will automatically load this file during compare for the selected dataset and use it as supplemental mapping.
